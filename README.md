# Alice-in-Wasteland

A 2D roguelike bullet hell survival game built in Unity 6, developed as a graduation thesis project focused on sustainable code architecture and performance optimization.

🎮 **[Play on itch.io](https://xxreator.itch.io/alice-in-wasteland)**

## About

Survive endless waves of enemies, collect experience to level up, and grow stronger as you fight your way through the wasteland. Inspired by the Vampire Survivors-style genre, the game focuses on chaotic enemy swarms, fast-paced combat, and scalable architecture.

## Note

Some optimization and debug tools (e.g. FPS counter) are visible in the settings menu — these are present for thesis evaluation purposes and can be disabled for a cleaner playthrough.

## Built With

- Unity 6
- C#
